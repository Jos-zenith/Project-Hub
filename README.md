# 🌐 Zenith Joshua — Portfolio

A personal portfolio website showcasing my journey as an IT student, entrepreneur, and problem-solver.  
Built with modern web technologies to highlight projects, skills, and community impact.

---

## 🚀 Features
- **Hero Section**: Name, tagline, and quick links (LinkedIn, GitHub, Resume).
- **About Me**: Short bio with cultural + professional context.
- **Projects**: Case studies with Problem → Solution → Tech Stack → Impact.
- **Experience Timeline**: Interactive timeline of leadership, internships, and startups.
- **Skills**: Tech stack grid + soft skills visualization.
- **Community Impact**: Initiatives focused on South Indian challenges.
- **Blog**: Insights on tech for local problems (English + Tamil).
- **Contact**: Clean form + social links.

---

## 🛠️ Tech Stack
**Frontend**
- [Next.js](https://nextjs.org/) — React framework for fast, SEO-friendly pages
- [Tailwind CSS](https://tailwindcss.com/) — utility-first styling
- [Framer Motion](https://www.framer.com/motion/) — animations and transitions
- [Chart.js](https://www.chartjs.org/) / [D3.js](https://d3js.org/) — data visualization
- [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction) — 3D/IoT visualizations

**Backend**
- [Node.js](https://nodejs.org/) + [Express](https://expressjs.com/) — lightweight backend
- [MongoDB](https://www.mongodb.com/) / [Firebase](https://firebase.google.com/) — data storage
- [Sanity.io](https://www.sanity.io/) / [Strapi](https://strapi.io/) — headless CMS for blog content

**Deployment**
- [Vercel](https://vercel.com/) — hosting and CI/CD
- [Cloudinary](https://cloudinary.com/) — image optimization
